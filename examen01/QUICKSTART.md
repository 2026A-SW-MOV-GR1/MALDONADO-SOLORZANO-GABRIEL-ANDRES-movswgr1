# ⚡ INICIO RÁPIDO

## 1. Instalar Dependencias

```bash
cd examen01
npm install
```

## 2. Ejecutar en Web

```bash
npm run web
```

Se abrirá automáticamente: `http://localhost:19006`

## 3. ¡Usar la App!

- ✍️ Rellena el formulario y crea items
- 🔄 Haz clic en "Alternar Almacenamiento" para cambiar de BD
- 🗑️ Elimina items con el botón de eliminar
- 📊 Mira cómo los datos cambian según el almacenamiento activo

## Otros Comandos

```bash
npm test           # Ejecuta pruebas
npm run lint       # Verifica código
npm run verify     # Valida configuración
```

## Requisitos

✅ Node.js 14+
✅ npm 6+
✅ Navegador moderno
